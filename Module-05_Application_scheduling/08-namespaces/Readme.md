# Understanding `Namespaces`

## Initial Namespaces

- Default
- kube-node-lease
- kube-public

### Step-01: List Namespaces

- Run following command to list all the namespaces present in your K8s cluster:

  ```
  kubectl get namespaces

  OR

  kubectl get ns
  ```
